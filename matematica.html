A mediana é uma medida estatística que representa o valor central de um conjunto de dados ordenados. Para calculá-la, basta organizar os dados em ordem crescente ou decrescente e encontrar o valor que está no meio. Se o conjunto tiver um número par de elementos, a mediana é a média dos dois valores centrais. A fórmula para a mediana, M, é:
Mediana=X( 2n+1)

A média é uma medida estatística que representa o valor central de um conjunto de dados. É calculada somando todos os valores do conjunto e dividindo pela quantidade de elementos. A média é uma medida de tendência central que proporciona uma visão geral do conjunto de dados, mas pode ser sensível a valores extremos, conhecidos como outliers.

A moda é outra medida estatística que descreve a tendência central de um conjunto de dados. Ao contrário da média, que representa o valor médio, a moda refere-se ao valor que ocorre com maior frequência no conjunto de dados. Pode haver uma moda (unimodal), várias modas (multimodal) ou nenhum valor que se repita (conjunto de dados amodal)
